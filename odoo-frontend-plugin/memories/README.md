# Odoo Frontend Plugin - Memories

This folder contains persistent knowledge and best practices for Odoo frontend development.

## Files

| File | Description |
|------|-------------|
| `theme_development.md` | Theme creation patterns and best practices |
| `publicwidget_patterns.md` | publicWidget JavaScript patterns |
| `scss_variables.md` | SCSS variable reference and color system |
| `bootstrap_mapping.md` | Odoo version to Bootstrap version mapping |
| `figma_extraction.md` | Figma design variable extraction patterns |

## Usage

These memories are automatically loaded when the odoo-frontend skill is activated, providing context for:
- Theme scaffolding decisions
- Code pattern selection
- Version-specific implementations
- Design system integration
