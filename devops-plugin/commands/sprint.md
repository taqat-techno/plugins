# /sprint - Sprint Progress Summary

Parse `$ARGUMENTS` for `--full` flag.

**Standard**: Work items by state, velocity, burndown, blockers.
**Full** (`--full`): Adds builds, test results, PRs, security alerts.

Use the devops skill for:
- Sprint query patterns (WIQL for current iteration)
- Work item state aggregation
- Velocity calculation methodology
- Burndown chart data extraction
- Build/test/PR integration (full mode)
- Report formatting and presentation
