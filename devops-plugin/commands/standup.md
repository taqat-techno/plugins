# /standup - Daily Standup Notes

Generate standup notes from Azure DevOps work items.

Use the devops skill for:
- WIQL query for user's recent work items
- "Yesterday/Today/Blockers" format
- State change detection (what moved since yesterday)
- Active work item summarization
- Blocker identification and escalation suggestions
